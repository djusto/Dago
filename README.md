# Dicas de como usar git
In the linux folder:
```
> git clone https://github.com/djusto/ExData_Plotting1
> cd ExData_Plotting1
```
Create or move new files to this folder
```
> vi newfile.txt
> git status
> git add newfile.txt
> git status
> git commit  (mensagem)
> git status
> git push   # Finally sending
```
